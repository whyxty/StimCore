# Расчёт СКО (солянокислотной обработки)

Веб-приложение на Streamlit для проектирования солянокислотной обработки нефтегазовых скважин по методике Приложения В (ветка СКО — без HF/ГКО).

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется в браузере по адресу http://localhost:8501.

## Тесты

```bash
pytest tests/
```

## Профиль месторождения

Все коэффициенты, специфичные для региона, вынесены в `config.json` и редактируются в разделе **«Настройки месторождения»**:

- α в формуле `k_г.о = exp(−α·r)` — Предкарпатье: 0,1
- a, b в `ΔG_s = a·C_гл + b·C_к` — 0,25 / 0,5
- Регрессии `k₀ = A·m₀^B` по 5 типам коллекторов (КЛ 1…5)
- `k_s* = A·exp(B·C_к)` — 0,9 / 0,2
- Удельные дебиты по диапазонам пористости (табл. В.2)
- Критерии отбора скважины: `m_пр`, `h_пр`, `C_к.пр`, `C_гл.пр`, `k_s.np`, `q_пр`
- Градиенты давления ГРП — для нефтяных и водонагнетательных скважин

Кнопки: **Сохранить профиль** / **Сбросить к Предкарпатью** / **Экспорт-Импорт JSON**.
